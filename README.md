# University Portal

A simple university website project with pages for home, admission, fees, and contact.

## Project structure

- `index.html` - homepage
- `admission.html` - admission form page
- `fees.html` - fee payment page
- `contact.html` - contact form page
- `css/` - stylesheets for each page
- `js/` - shared and page-specific JavaScript files
- `images/` - image assets used in the site

## Notes

- Navigation links and asset paths are updated to match the current project structure.
- Form actions use a simple JavaScript helper instead of backend email submission.
